House Rules · Why This Catalog Is Small

The bazaar's honesty policy.

RULE 01

Real repos only

Every entry links to a public, non-fork repository owned by IDC. Fork mirrors and aspirational products don't get stalls.

RULE 02

Numbers compute themselves

Stars and freshness come from the GitHub API in your browser. The entry count is the length of inventory.json. Nothing on this page is typed in by marketing.

RULE 03

Paid means fulfilled

A paid listing appears only when checkout, delivery, and metering all work end-to-end. Right now there is exactly one. That's the honest count.
