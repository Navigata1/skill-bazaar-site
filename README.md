# Skill Bazaar — the AgentPact skills layer

Truthful catalog of real IDC skills and frameworks. Every entry in [public/inventory.json](public/inventory.json) is a public, non-fork repo; the site computes its counts from that file and pulls stars/freshness live from the GitHub API. One paid listing (AgentPact Starter Key) with real end-to-end fulfillment.
